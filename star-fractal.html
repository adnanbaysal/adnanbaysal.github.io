How it Works

This generator uses the HTML5 Canvas API to render recursive geometric shapes in real-time.

The Algorithm

For every line segment, the code calculates n new branches originating from the end point separated with 360/n degrees each. Each new branch is scaled by ratio p. This process repeats until the depth limit is reached or the segment becomes too small to be visible.
